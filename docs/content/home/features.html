dashboard Content Manager

Simplifies site structuring and management, with content inclusion feature, content and style files can be conveniently separated.

Optimizes speed with lazy-loading feature, content will be loaded only when it actually comes into the user's screen view.

Can load content from different sites with just one HTML tag, making of reusability a native feature.

settings_ethernet Template Engine

Turns any HTML snippet into a template. Template preview is immediate and truly matching, NO custom tags or mustaches needed.

Maps data model's fields to view elements either automatically or by using custom binding-adapters, with full control over model-view transformation process.

Prevents style attributes redundancy hell with Scoped CSS files.

extension Component Manager

Empowers the implementation of agnostic component logic for creating components that can stick to any UI-framework and easily upgrade at any time.

Enables running of multiple components on the same UI view with components overlapping feature.

Introduces a new way of app theming with instant view switching, that allow replacing of the component's view template at any time.
