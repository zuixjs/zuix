### `ZxQuery` class *ZxQuery* is a *ZUIX* built-in class that implements a very lite subset of jQuery-like functionality. It can wrap elements in a DOM and provide useful methods for manipulating it. #### Constructor The constructor takes one optional argument that can be a DOM element, a node list or a valid DOM query selector string expression. If no parameter is given, the ZxQuery will wrap the root *document* element. 
var zxElement = new ZxQuery(element);
or 
var zxElement = zuix.$(element);
which is the recommended way of creating `ZxQuery` objects. #### Methods
